# Portfolio - Project Instructions

## Project Overview
Personal developer portfolio built with **Astro** and **React**.

## Tech Stack
- **Framework:** [Astro](https://astro.build/) (v5.7.0)
- **UI Library:** [React](https://react.dev/) (v19)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Components:** React components located in `src/components/`.
- **Layouts:** Astro layouts in `src/layouts/`.
- **Pages:** Astro pages in `src/pages/`.
- **Internationalization (i18n):** Configuration in `src/i18n/`.

## Deployment & Domain
- **Target Platform:** [Vercel](https://vercel.com/)
- **Custom Domain:** `josebordon.com`
- **Current Status:** Configured for Vercel deployment with `@astrojs/vercel` adapter.

## Development Guidelines
- Always use TypeScript for new components and logic.
- Follow the existing component structure in `src/components/`.
- Use Tailwind CSS for styling.
- Ensure any changes are compatible with Vercel deployment.
